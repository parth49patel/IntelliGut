# IntelliGut

> An AI-powered gut health companion built for the Apple Swift Student Challenge.

IntelliGut helps users understand the relationship between what they eat and how they feel — combining the power of on-device AI with an intuitive SwiftUI interface to deliver personalized gut health insights.

---

## 📖 Overview

Gut health is deeply connected to overall wellbeing, yet it remains one of the most overlooked areas of personal health. IntelliGut bridges that gap by using artificial intelligence to analyze dietary input and surface meaningful insights about gut health, food sensitivities, and nutritional patterns — all from the palm of your hand.

Built entirely in Swift as a submission for the **Apple Swift Student Challenge**, IntelliGut demonstrates how modern Apple frameworks can be combined to create a thoughtful, impactful health experience.

---

## ✨ Features

- **AI-Powered Food Analysis** — Leverages on-device machine learning to analyze meals and ingredients and surface gut health insights.
- **Intuitive SwiftUI Interface** — Clean, accessible UI designed with Apple's Human Interface Guidelines for a seamless experience.
- **Privacy-First Design** — All data stays on-device; no account required.

## 🚀 Getting Started

### Prerequisites

- Xcode 15 or later
- iOS 17+ / iPadOS 17+
- A Mac running macOS Ventura or later

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/parth49patel/IntelliGut.git
   ```

2. Open the project in Xcode:
   ```bash
   cd SSC
   open SSC.xcodeproj
   ```

3. Select your target device or simulator and hit **Run** (`⌘R`).
